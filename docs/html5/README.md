# Html5

10大新特性

html5总的来说比html4多了十个新特性，但其不支持ie8及ie8以下版本的浏览器

- 语义标签
- 增强型表单
- 视频和音频
- Canvas绘图
- SVG绘图
- 地理定位
- 拖放API
- Web Workers
- Web Storage
- Web Socket
