# 图像

在 HTML 中，图像由`<img />`标签定义

`<img />` 是单标签，意思是说，它只包含属性

要在页面上显示图像，你需要使用源属性(src)。src 指 "source"。源属性的值是图像的 URL 地址

**实例**

```html
<img src="url" alt="some_text" />
```

## 属性

|属性|描述|属性值|
|:---|:---|:---|
|src|文本|图像的路径|
|alt|文本|图像不能显示时的替换文本|
|title|文本|鼠标悬停时实现的内容|
|width|像素(XHTML不支持%页面百分比)|设置图像的宽度|
|height|像素(XHTML不支持%页面百分比)|设置图像的高度|
|border|数字|设置图像边框的宽度|
