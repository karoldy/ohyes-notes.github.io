# vim

Vim 掌握有五 (5) 个级别：

**0 级**：不了解 Vim
**1 级**：了解 Vim 基础知识
**2 级**：了解界面模式
**3 级**：了解各种动作
**4 级**：不需要界面模式

## 3.配置

.vimrc是vim的配置文件

/etc/vim/vimrc 是所有用户都共用的配置

例如：设置语法高亮，设置显示行号

vim ~/.vimrc

```shell
synctx on
set number
```
