## 效果演示

animate.css的使用非常简单，因为它是把不同的动画绑定到了不同的类里，所以想要使用哪种动画，只需要把通用类animated和相应的类添加到元素上就行了

### 【Attention(晃动效果)】

```
bounce
flash
pulse
rubberBand
shake
headShake
swing
tada
wobble
jello
```

### 【bounce(弹性缓冲效果)】

```js
bounceIn

bounceInLeft
bounceInRight
bounceInUp
bounceOut
bounceOutDown
bounceOutLeft
bounceOutRight
bounceOutUp
```
